# Deep Vision App

Bu proje, YOLOv8 ve DeepSORT kullanarak inşaat alanlarında **baret**, **sigara** ve **telefon** gibi nesnelerin tespitini ve takibini yapmaktadır.

## 📦 Özellikler
- YOLOv8 ile nesne tespiti
- DeepSORT ile çoklu nesne takibi
- Gerçek zamanlı video analizi
- Python tabanlı geliştirme
